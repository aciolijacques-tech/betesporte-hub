# 03 — Carteira de Influenciadores

> **Contexto:** Dados sobre influenciadores, perfis, plataformas e gestão.
> **Leia quando:** A pergunta envolver influenciadores, carteira, nichos, engajamento, cachê ou plataformas.

---

## Visão geral da carteira

| Dado | Valor |
|---|---|
| Total de influenciadores | 63 |
| Macro influenciadores | ~18 (estimativa inicial) |
| Micro influenciadores | ~45 (estimativa inicial) |
| Status de gestão | Em estruturação — sem classificação prévia |
| Gestor responsável | Uma pessoa específica (gestor de influenciadores) |

---

## Plataformas ativas

- Instagram
- TikTok
- YouTube
- Twitter/X

---

## Classificação por porte

| Porte | Descrição |
|---|---|
| **Macro** | Influenciadores de grande alcance — tipicamente 100k+ seguidores |
| **Micro** | Influenciadores de nicho — tipicamente até 100k, alto engajamento |

*Antes do hub, os influenciadores não eram classificados por nicho, público ou qualquer outro critério além do porte.*

---

## Campos do perfil de cada influenciador

| Campo | Visibilidade |
|---|---|
| Nome / @ | Todos |
| Porte (macro/micro) | Todos |
| Nicho | Todos |
| Público principal | Todos |
| Plataformas | Todos |
| Seguidores totais | Todos |
| Taxa de engajamento (%) | Todos |
| Alcance médio por post | Todos |
| Status (ativo/negociação/inativo) | Todos |
| Responsável interno | Todos |
| Contato (WhatsApp/email) | Todos |
| Cachê/mês (R$) | **Somente nível 3 ou acima** |
| Observações | Todos |
| Histórico de campanhas | Todos |

---

## Filtros disponíveis na gestão

- Por porte: Macro / Micro
- Por plataforma: Instagram / TikTok / YouTube / Twitter/X
- Por status: Ativos
- Busca livre: nome, nicho, público, responsável

---

## Status possíveis de um influenciador

- **Ativo** — parceria em vigor
- **Em negociação** — contrato em discussão
- **Inativo** — parceria encerrada

---

## Dados a cadastrar quando houver

Quando o usuário quiser cadastrar influenciadores reais, solicitar no formato:
```
@nome | macro ou micro | nicho | público | plataformas (ig,tk,yt,tw) | seguidores | engajamento% | responsável
```
