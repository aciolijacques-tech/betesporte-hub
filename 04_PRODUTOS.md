# 04 — Catálogo de Produtos

> **Contexto:** Produtos e linhas da BETesporte.
> **Leia quando:** A pergunta envolver produtos, linhas, categorias ou canais de venda.

---

## Status atual

O catálogo de produtos está **aguardando cadastro**.

O usuário informou que preencherá os produtos reais quando o hub estiver pronto. Nenhum produto foi cadastrado ainda.

---

## Tipo de negócio

A BETesporte trabalha com **múltiplos produtos e linhas** — empresa de apostas esportivas com produtos digitais.

---

## Campos do perfil de cada produto

| Campo | Visibilidade |
|---|---|
| Nome do produto / linha | Todos |
| Categoria | Todos |
| Público-alvo | Todos |
| Canais de venda | Todos |
| Status | Todos |
| Faixa de preço | **Somente nível 3 ou acima** |

---

## Status possíveis de um produto

- **Ativo** — disponível no mercado
- **Lançamento** — em fase de lançamento
- **Descontinuado** — fora de linha

---

## Canais de venda esperados

Baseado no perfil da empresa (apostas esportivas / digital):
- App mobile
- Site / plataforma web
- Possivelmente loja física (a confirmar)

---

## Quando o usuário quiser cadastrar produtos

Solicitar no formato:
```
Nome | Categoria | Público-alvo | Faixa de preço | Canais de venda | Status
```
